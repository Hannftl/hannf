Langkah 1 : Memahami Cara Kerja Flip Flop

  • JK Flip Flop:
    • J=1, K=0 → Set (Q=1)
    • J=0, K=1 → Reset (Q=0)
    • J=K=0 → Hold
  • D Flip Flop: Output selalu mengikuti input D saat clock aktif.
  • Konversi JK ke D Flip Flop: J=D, K=0
    • Jika D=1 → J=1, K=0
    • Jika D=0 → J=0, K=1

Langkah 2 : Analisis Setiap Clock Pulse Secara Berurutan

  • Clock 1: D=1 → J=1, K=0 (SET)
    • Q(t) = 0 (kondisi awal)
    • Q akan di-SET menjadi 1
    • Q(1) = 1
  • Clock 2: D=0 → J=0, K=1 (RESET)
    • Q(t) = 1 (hasil dari clock 1)
    • Q akan di-RESET menjadi 0
    • Q(2) = 0
  • Clock 3: D=1 → J=1, K=0 (SET)
    • Q(t) = 0 (hasil dari clock 2)
    • Q akan di-SET menjadi 1
    • Q(3) = 1
  • Clock 4: D=1 → J=1, K=0 (SET)
    • Q(t) = 1 (hasil dari clock 3)
    • Q tetap bernilai 1
    • Q(4) = 1

Langkah 3 : TABEL HASIL TRACKING

Langkah 4 : Kesimpulan

HASIL AKHIR:
Setelah Clock 1: Q = 1
Setelah Clock 2: Q = 0
Setelah Clock 3: Q = 1
Setelah Clock 4: Q = 1 (tetap)
Output akhir Q = 1
